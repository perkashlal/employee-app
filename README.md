# Employee Scheduling System
Description
The Employee Scheduling System is a Java 17 application for managing workforce data using MongoDB. Built with an MVC architecture, it separates a Swing-based GUI from core business logic. The project demonstrates high software quality standards by maintaining 100% test coverage by following TDD approach, integration testing with Testcontainers, and mutation testing using Pitest.
